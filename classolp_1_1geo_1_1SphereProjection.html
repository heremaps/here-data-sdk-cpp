Detailed Description

Projection of geo coordinates onto a sphere using earth radius.

The sphere projection defines Earth Centric Earth Fixed coordinate system ECEF http://en.wikipedia.org/wiki/ECEF by projecting geo coordinates onto a sphere with radius equal to earth radius - the unit of world coordinates is meter.

The geo coordinates map to ECEF axes as follows: lat = 0, long = 0 -> X lat = 0, long = 90 -> Y lat = 90, long = 0 -> Z
