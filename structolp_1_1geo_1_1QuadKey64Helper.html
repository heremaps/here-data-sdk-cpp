Detailed Description

The QuadKey64Helper struct is a helper structure for basic operations on 64 bit morton quad keys.

This class can be used to prevent conversions between tile keys and quad keys for basic operations.

◆ GetSubkey()

QuadKey64Helper olp::geo::QuadKey64Helper::GetSubkey ( int  delta) const

Returns a sub quad key that is relative to its parent.

This function can be used to generate sub keys that are relative to a parent that is delta levels up in the sub tree.

This function can be used to create shortened keys for quads on lower levels if the parent is known.

Note - the sub quad keys fit in a 16 bit unsigned integer if the delta is smaller than 8. If delta is smaller than 16, the sub quad key fits into an unsigned 32 bit integer.

Parameters
deltathe amount of levels relative to its parent quad key. Must be greater or equal to 0 and smaller than 32.
Returns
The quad key relative to its parent that is delta levels up the tree.
