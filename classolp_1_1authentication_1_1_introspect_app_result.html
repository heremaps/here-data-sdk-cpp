◆ GetReferrers()

const std::vector<std::string>& olp::authentication::IntrospectAppResult::GetReferrers ( ) const
inline

List referrer urls to register/registered with Application.

On create/update: if the parameter is not present, the current value(s) are unchanged. If empty list specified, current value(s) will be reset to empty list. Value has a min length of 1 char and max of 255 chars. Wildcards are NOT allowed. The only valid characters: alphanumerics, '-', '_', '.', '/' The protocol is NOT specified, i.e. no http:// or https:// Some examples: here.com, localhost, 127.0.0.1, www.facebook.com/hello/world/.

Returns
The list referrer urls to register/registered with Application.
